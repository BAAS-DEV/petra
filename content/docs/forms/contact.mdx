## Overview

Records from inputs on your "Contact Us" form.

## Client Side

The public can view this form at `/contact`

![Client Side Contact](/docs/forms/contact/form.png)

### Viewing Form Submissions

Each record records the following:

| Attribute Name |
| -------------- |
| Name           |
| Email          |
| Message        |
| Created At     |

Navigate to `/admin/forms`, or use the admin navigation

![Viewing Contact Image](/docs/forms/nav.png)

You will find a table. Please make sure you have the **_Contact Form_** tab selected.

Each record will have a **_View_** button. Click that to view the record.

![Viewing Contact Image](/docs/forms/contact/table.png)

![Viewing Contact Image](/docs/forms/contact/view.png)

### Deleting Records

You can select multiple rows of contact submissions at a time, by **Pressing the Select Row Icon**, at which point the table row will reflect that you selected it. You can now hit the **Delete Button At the Bottom Of The Table**

![Deleting Resource Image](/docs/forms/contact/delete.png)
