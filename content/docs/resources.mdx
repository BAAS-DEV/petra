## Overview

Resources are downloadable or otherwise viewable files, links, items, etc that are publically available.

## Client Side

The public can view these at `/resources`

![Client Side Resources](/docs/resources/overview.png)

## Resource Attributes

| Attribute Name |
| -------------- |
| Title          |
| Order          |
| Link           |
| Icon Link      |

### Viewing Resources

Navigate to `/admin/resources`.

On this page you will find a table with the attributes listed for **Resources**

![Viewing Resources Image](/docs/resources/view.png)

### Creating Resources

Locate the "Create" Button, at the **Top Right of the Table Header**.

![Create Resource Image](/docs/resources/create.png)

This will open up a form, allowing you to fill in all information. Please refer to the attributes table for what each of these fields represents.

![Create Form Resource Image](/docs/resources/createForm.png)

### Editing Resources

Every row entry will have an **Edit** Button. Press this to open up the editing form.

![Edit Team Image](/docs/resources/edit.png)
